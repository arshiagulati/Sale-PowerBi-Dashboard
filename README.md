# Sales Dashboard (2020–2022) — Power BI

An interactive **Power BI Sales Dashboard** providing a comprehensive view of sales performance from **2020 to 2022**. The dashboard helps stakeholders track revenue, profit, quantity, costs, and brand/category performance across India.

---

## Overview

This dashboard is designed for **business analysis and decision-making**, offering insights into:

- Sales trends
- Product category performance
- Brand contribution
- Geographic distribution
- Supervisor-wise filtering

---

## Key Metrics

- **Total Sales:** 99M
- **Total Cost:** 76M
- **Total Profit:** 23M
- **Total Quantity Sold:** 13K
- **Total Transactions:** 5,095

---

## Visuals Included

- **Quantity by Category (Bar Chart)**  
  Monitor, Mouse, CPU, Graphic Card, SSD, RAM, etc.

- **Sales by State (Map Visual)**  
  Geographic distribution of sales across India.

- **Total Sales by Brand (Pie Chart)**  
  Intel, Samsung, Dell, Nvidia, Western Digital, Acer, and more.

- **Supervisor Slicer**  
  Filter dashboard data by individual supervisors.

---

## Tools & Technologies

- **Power BI Desktop**
- **DAX Measures**
- **Interactive Filters & Slicers**

---

## How to Use

1. Download the `.pbit` file.
2. Open it in **Power BI Desktop**.
3. Use slicers (Supervisor, Category, Brand) to explore insights.
4. Publish to **Power BI Service** for sharing (optional).
